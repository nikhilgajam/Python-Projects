PyTyping is a simple and intuitive typing speed analysis program.

Time Box: You can type only numbers in time box and you can press enter key to focus on input box.

Special Keys:
Enter = You can press enter to start typing when focused on text input box.
Esc   = You can press this key to change the displayed text before starting the timer. 
        If timer is started then it acts like halt typing. key.

You can see the realtime wpm and other info when you start typing.
