Tap Code is a morse like code used to convey message using sounds

Tap Code Symbols Meaning:
.  = (Dot) It is a tap
   = (Space) It is there to differentiate column from row
/  = (Slash or forward slash) It is used to differentiate two different letters 
// = (Double slash or double forward slash) It is used as space between two words


You can use tap code visualizer to understand tap code more.
